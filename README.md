# NewsGuard-An-Intelligent-System-for-Detection-of-Misinformation-in-News-Media
NEWSGUARD — Hybrid ML + Deep Learning fake news detection system (TF-IDF + Sentence Embeddings) achieving 99.7% accuracy, with CNN, LSTM, SVM baselines and a real-time Streamlit web app for live news analysis. MCA Final Year Project, Sharda University 2026.
